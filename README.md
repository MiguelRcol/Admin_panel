# Admin Dashboard

A responsive-style admin dashboard layout built with HTML and CSS Grid as part of The Odin Project curriculum.

## Built with

- HTML5
- CSS3
- CSS Grid
- Flex/Grid nested layouts
- SVG icons
- Google Fonts
- Git & GitHub Pages

## What I practiced

- Creating a full-page dashboard layout
- Using CSS Grid for main and nested sections
- Organizing assets and icons
- Building reusable cards
- Improving visual hierarchy and spacing
